# TPMS结构参数探索器体验说明

## 打开方式

解压后使用最新版 Chrome 或 Edge 打开 `index.html`（落地页），点击"开始探索"即可进入 `app.html` 主应用。

## 注意事项

- 首次打开需要联网加载 Three.js CDN。
- 视口支持拖拽旋转、滚轮缩放、右键平移。
- 顶部工具按钮包含：复位视角、自动旋转、复制当前参数链接、保存 PNG、导出 STL（二进制格式，可直接用于 3D 打印）和全屏。
- 右侧控制面板的各分区均可折叠，方便聚焦当前调节内容。
- 视口左上角的状态 chip 默认收起，点击可展开查看完整工程指标（估算孔隙率、材料占比、顶点数、三角面数、重建耗时等）。
- 如果浏览器拦截本地文件下载，请允许页面下载 STL / PNG 文件。

## 推荐体验流程

1. 首次进入会自动弹出 5 步新手引导；之后可随时点击顶栏"新手引导"重新查看。
2. 切换 Gyroid、Diamond、Schwarz P、Neovius、I-WP、F-RD 六类曲面，并在"当前曲面隐函数"区逐项拖动公式权重滑块，观察每个数学项如何塑造曲面形态。
3. 调节孔隙率、单元密度、壁厚系数，观察结构实时重建与估算孔隙率变化。
4. 拖动截面扫描滑块，剖开结构观察内部连通孔道。
5. 在"结构拓扑"中切换 实体网络 / 等厚双壳 / 梯度双壳（功能梯度），并在立方体 / 圆柱体两种容器之间切换。
6. 点击仿生骨支架、轻量化零件、散热结构三个预设，留意弹出的"为什么这样配"场景卡。
7. 悬停带虚线下划线的术语（孔隙率、等值常数 C、权重等），查看对应的解释卡。
8. 复制当前参数链接分享给他人，或使用 PNG / STL 导出当前结果。
